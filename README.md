# ♟ Шашки-бот для Telegram

Полноценный бот для игры в шашки в групповых чатах Telegram с таблицей лидеров.

## Возможности
- 🎮 Игра в шашки вдвоём прямо в групповом чате
- ♟ Полные правила: обязательное взятие, дамки, многократные прыжки
- 🟢 Подсветка допустимых ходов
- 🏆 Таблица лидеров (SQLite) с рейтингом побед
- ⏰ Автоотмена вызова через 5 минут
- 🏳️ Кнопка сдаться прямо в игре

## Установка

### 1. Создай бота
1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Напиши `/newbot` и следуй инструкциям
3. Скопируй полученный токен

### 2. Настрой права бота в группе
В BotFather: `/mybots` → твой бот → Bot Settings → Group Privacy → **отключи** (Disable)
Это нужно, чтобы бот видел команды в группе.

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Укажи токен
Замени `YOUR_BOT_TOKEN_HERE` в `bot.py` на свой токен:
```python
BOT_TOKEN = "123456789:ABCDefgh..."
```
Или установи переменную окружения:
```bash
export BOT_TOKEN="твой_токен_здесь"
python bot.py
```

### 5. Запуск
```bash
python bot.py
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/play` | Бросить вызов на партию |
| `/accept` | Принять вызов |
| `/resign` | Сдаться |
| `/leaderboard` | Таблица лидеров |
| `/rules` | Правила игры |

## Как играть
1. Добавь бота в группу
2. Один игрок пишет `/play` — бросает вызов
3. Другой игрок пишет `/accept` — игра начинается
4. Нажми на свою шашку (⚫/⚪) — появятся зелёные точки 🟢
5. Нажми на зелёную точку — шашка переместится
6. Дамки: ♟ достигает последнего ряда → становится 👑/🔱

## Правила
- Чёрные (⚫) ходят первыми
- Взятие **обязательно**, если есть возможность
- При цепочке взятий тот же игрок бьёт дальше
- Дамки ходят на любое расстояние по диагонали
- Побеждает тот, кто съел все шашки или заблокировал соперника
